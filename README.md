# Proyecto Autos

Este es un proyecto para gestionar y visualizar información relacionada con autos. 

## Características

- CRUD autos.
- Visualizar detalles de cada auto.

## Instalación

1. Clonar este repositorio.
2. Instalar SDK 6.0.425 y RUNTIME 6.0.33
